Über den Java Taschenrechner

In diesem Projekt habe ich mich mit Desktop Applikationsentwicklung beschäftigt. Ich habe das JavaFX Framework in Kombination von klassischem Java 23 genutzt. Ich habe davor noch keine Erfahrung mit Desktop Development in Java gehabt. Dieses Projekt hat mir ebenfalls stark bei dem Ausbau meiner CSS-Fähigkeiten geholfen.
